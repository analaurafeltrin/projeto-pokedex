Pokédex 🔴

Desenvolvi uma Pokédex com HTML, CSS e JavaScript puro, que consome a PokéAPI para listar Pokémons com carregamento dinâmico.

> Projeto desenvolvido durante o curso de JavaScript da [DIO (Digital Innovation One)](https://www.dio.me/), com foco na prática de consumo de APIs REST.

## 🚀 Acesse o projeto

🔗 [Link do Site](https://analaurafeltrin.github.io/projeto-pokedex/)

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- [PokéAPI](https://pokeapi.co/) — API pública REST

---

## 📚 O que foi praticado

- Consumo de API REST com `fetch` e `Promises`
- Manipulação do DOM com JavaScript puro
- Estruturação semântica com HTML5
- Estilização responsiva com CSS3
- Organização de projeto front-end sem frameworks

---

## 💡 Aprendizados

Este foi meu **primeiro projeto utilizando uma API externa**. Sinto que meu maior desafio foi entender o fluxo assíncrono das requisições do JS, precisando entender porque os dados não estavam chegando da maneira que eu esperava. Quando deu tudo certo, foi muito satisfatório :)
